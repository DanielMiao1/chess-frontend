An open-source and completely free-to-use chess website

Start a game with the popular chess engine Stockfish, ported to JavaScript (v15). Or, generate a game link to share and start a game with other people.

Use this tool to evaluate and find the best move of a position. Positions can be imported in PGN or FEN format, or simply by playing moves leading to the position.

This chess website, made with Node.js (express) and HTML, is completely free-to-use, and is hosted mainly on Vercel. DDoS mitigation, caching, and (reverse-)proxying for this website is provided by Cloudflare. This website uses the chess.js library for PGN and FEN importing, move validation, and other board-related logical operations.
